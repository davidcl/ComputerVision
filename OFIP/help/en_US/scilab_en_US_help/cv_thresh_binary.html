cv_thresh_binary

Binary Thresholding

Calling Sequence

image_ret =cv_thresh_binary(image)

Arguments

image

Image that needs to be thresholded

image_ret

Thresholded Image is returned to output variable "image"

Description

Provides Binary thresholding

Thresholding is the simplest method of image segmentation. From a grayscale image, thresholding can be used to create binary images . This function takes an image as input and thresholds the image. the thresholded image is returned to User via output variable

Examples

image_ret=cv_thresh_binary(input_image)
